1. 전체 계좌 목록 조회
   -> 체크카드와 연동되어있는 결제계좌(입출금 계좌), 챌린지용 적금 계좌 확인 용도
     -> 구분(계좌 종류), 계좌번호, 잔액 등 확인 가능

2. 계좌 거래내역 조회
   -> 커피를 사마시면 입출금 계좌에서 결제금액이 출금되므로, 거래내역 조회를 통해 커피 소비 여부 확인 용도
    -> 거래일자, 출금금액, 내용(스타벅스 있음!!) 등 확인 가능

3. 예적금계좌 내역 및 상세 조회
    -> 커피 소비가 확인되면, 챌린지와 연동된 적금 계좌에 입금 여부 확인 용도
    -> 적금 계좌에 입금이 되면 챌린지 성공, 입금되지 않으면 실패
    -> 계좌번호, 이율, 납입횟수, 미납회차, 거래내역(거래일자, 입금 등) 등 확인 가능

4. 이체(원화)
    -> 커피 소비 후, 어플 화면에 이체 버튼이 생성되고, 누르면 입출금 계좌에서 적금 계좌로 이체
    -> 출금계좌번호, 입금계좌번호, 이체금액, 표시 메모 내용 설정 가능

5. SOL 앱푸시
    -> 커피 소비 시, 적금 계좌 입금 성공 시 메시지 전송
    -> 고객번호, 메시지 내용 설정 가능
